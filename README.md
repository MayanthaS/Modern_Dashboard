# Modern Dashboard

A responsive personal finance dashboard built with React and Vite.

This project focuses on a clean blue-themed UI for tracking key financial information in one place, including cards, transactions, reports, budget planning, subscriptions, savings, loans, and financial goals.

## Features

- Responsive dashboard layout for desktop, tablet, and mobile
- Sidebar navigation with finance-related sections
- Top content bar with menu, search, and notifications
- Summary cards area for quick account overview
- Transactions panel with recent activity
- Reports chart with monthly bar visualization
- Budget widget with categorized spending items
- Subscriptions widget with active plans and due dates
- Savings widget with progress tracking
- Loans widget with circular repayment progress
- Financial goals widget with progress bars and targets

## Tech Stack

- React 19
- Vite
- CSS
- ESLint

## Getting Started

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

### Preview production build

```bash
npm run preview
```
